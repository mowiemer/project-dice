# Devlog

Welcome to the devlog for nicedice.net!

## About

This page houses the things I learned on this web dev journey of nicedice.com, with commentary. Established in December 2025.

The purpose of the website is to practice web development for a project that I am interested in, simply to build experience and have fun. The knowledge can then be transferred to any future project.

The purpose of this devlog is:

- To hold myself accountable to inspire progress
- Have a running list of reflections and thoughts
- Maybe inspire any developer who sees this to pull the trigger on starting their own website

At the time of writing (25.12.2025), I have minimal proficiency in web development. I have a minor in computer sciences, have participated in a couple hackathons, and know the basics of html, css, and js. I have exposure to d3 and a linkedIn certificate or two of the "essential web dev" trainings, but have never fleshed out any sort of website.

I have several personal development goals for this project:

- To learn about css and javascript ui elements using React and associated populater packages/frameworks.
- To learn about data visualizations
- To learn about responsive UI design and snappy animations
- To learn about marketing, and soliciting/incorporating user feedback
- To learn about logo design
- To learn how to navigate Git and Github more efficiently
